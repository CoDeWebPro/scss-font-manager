Локальный шрифт (симуляция)

Этот текст использует локальный шрифт "Custom Font" с fallback на sans-serif. В реальном проекте здесь был бы подключен локальный файл шрифта.

@include font-face( $family: "Custom Font", $file: "custom-font", $path: "/fonts/", $weight: "400", $style: "normal", $fallback: sans-serif, $local: "Custom Font Local" );

Проверка font-display

Откройте DevTools → Network → перезагрузите страницу, чтобы увидеть как загружаются шрифты с font-display: swap

// В DevTools вы увидите: // font-display: swap; // для Google Fonts // font-display: swap; // для локальных шрифтов (из $font-display-default)

Проверка unicode-range

В DevTools → Elements найдите @font-face правила и проверьте unicode-range для разных subset:

// Roboto (cyrillic): unicode-range: U+0400-04FF, U+0500-052F, U+2DE0-2DFF, U+A640-A69F; // Open Sans (latin): unicode-range: U+0000-00FF, U+0100-017F, U+0180-024F, U+1E00-1EFF;
